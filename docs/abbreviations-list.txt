MANAGEMENT COMMANDS
===================
abbr_add <name> <expansion>    Add custom abbreviation
abbr_rm <name>                 Remove abbreviation
abbr_list [filter]             List all abbreviations
abbr_search <term>             Search abbreviations
abbr_help                      Show comprehensive help
abbr_stats                     Show statistics
abbr_tips                      Show random tip
abbr_cheatsheet                Quick reference card

USAGE
=====
1. Type abbreviation
2. Press SPACE to expand
3. Edit if needed
4. Press ENTER to execute

EXAMPLES
========
gst                    → git status
gco -b feature         → git checkout -b feature
dcu -d                 → docker-compose up -d
kgp -n production      → kubectl get pods -n production
