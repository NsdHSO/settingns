# Zsh Configuration Cheatsheet

## Git Aliases
- `gis` - git status
- `gid` - git diff
- `gdd` - git add
- `gip` - git push
- `gic` - git cz (commitizen)
- `gim` - git commit -m
- `gio` - git

## Git Functions
- `gc [args]` - git commit
- `giofetch` - git fetch
- `giopull` - git pull
- `gtore` - git restore
- `ghard` - git reset --hard
- `ginit` - git init

## Utility Functions
- `killport <port>` - Kill process on specific port
- `nxg` - Custom dev workflow

## Yarn Aliases/Functions
- `yas` - yarn
- `ylock` - yarn lock
- `yall` - yarn all
- `yaw` - yarn add

## fzf Shortcuts
- `Ctrl+R` - Search command history
- `Ctrl+T` - Search files
- `Alt+C` - Search directories

## zoxide
- `z <partial-name>` - Jump to directory
- `zi` - Interactive directory picker

## iTerm2 Hotkeys
- `⌘+\`` - Toggle hotkey window
- `⌘+D` - Split pane vertically
- `⌘+Shift+D` - Split pane horizontally
- `⌘+[` / `⌘+]` - Navigate panes
- `⌘+T` - New tab
- `⌘+W` - Close tab/pane

## Powerlevel10k
- `p10k configure` - Reconfigure prompt
- `p10k reload` - Reload configuration

## General Aliases
- `vim` - nvim
- `python` - python3
- `pn` - pnpm
- `la` - ls -la
- `work®` - cd to Work/financial_health
